# Board of Directors — Deliberation Log

This file maintains a historical record of all council deliberations. Each entry captures who was summoned, what was decided, and the outcome.

---

## Log Format

Each deliberation is recorded with the following structure:

```markdown
### [YYYY-MM-DD] Deliberation #[N]: [Brief Title]

**Decision**: [The question posed to the council]

**Panel**:
- [Agent 1] — [Role]
- [Agent 2] — [Role]
- [Agent 3] — [Role]

**Arbiter**: Marcus Aurelius

**Verdict Summary**: [1-2 sentence summary of Marcus Aurelius's recommendation]

**Recommendation**: [Proceed / Do Not Proceed / Proceed with Conditions / Defer]

**Confidence**: [High / Medium / Low]

**Outcome** *(updated later)*: [What actually happened]

**Notes**: [Any relevant observations]

---
```
